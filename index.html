Barb% is an unofficial speedrun category for Enter The Gungeon that requires clearing all floors (including secret floors) and beating all bosses. This includes beating all three rounds of Punch-Out with the Resourceful Rat, however it does not include High Dragun.

This run was originally conceived by BarbarousKing for fun, but has since enjoyed modest popularity in his community. This is an unofficial category, so please do not ask leaderboard moderators to add this as an official category.
